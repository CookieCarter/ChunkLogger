## Chunk Logger

A simple mod which will log bedrock (on y-levels 4 and 123) in the Nether to a file. Meant to be used in conjunction with a [bedrock seed cracker](https://github.com/19MisterX98/Nether_Bedrock_Cracker). (Said seed cracker only supports importing up to 200 blocks.)

### Usage

To use just type '/logbedrock' while in the Nether.

You can also specify a radius as an optional parameter (e.g. '/logbedrock 10'). By default the command has a radius of 8 blocks.
> [!WARNING]
> Entering large values will cause the game to hang.

For the best results with the seed cracker, input a radius that gives ~200 blocks.

The file will be saved in the root directory of your minecraft instance.
